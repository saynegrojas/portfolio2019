my projects




tower finder.



content:

web development.

started:

june 4th, 2019

completed:

june 15th, 2019


Tower Finder is an app that allows developing wireless sites to locate existing nearby towers. It is motivated towards business to give them the option of working with other cellular site owners in using their towers. This app takes a specific location by inputing a longitude and latitude. This was done by using Google Maps Javascript API and Geolocation. Also, Google Maps was implemented along with markers and info windows. As for the data, we used Firebase to store data with the location of the towers, and its specific information, and extracted data using AJAX. Lastly, Tower Finder was created in HTML, CSS, and Javascript. GitHub: Tower Finder




drink properly.



content:

web development.

started:

August 1st, 2019

completed:

August 10th, 2019


The purpose of Drink Properly app is to locate the nearest bar and restaurant with their happy hour specials. The way it works is simply type a zip code in the search field. We used Google Maps Javascript API and Geolocation, plus its markers and info windows. Moreover, Drink Properly was built in Javascript, CSS, and HTML for the frontend, the client-side. For the backend, the server-side, my team and I used Node.JS and Express to access our database. Also, we used a relational database in MYSQL to create and store our data. GitHub: Drink Properly




know me.



content:

web development.

started:

September 17th, 2019

completed:

October 4th, 2019


KnowMe is a game application. The objective of the game is to see if your trivia knowledge is better than others. It’s a multiplayer game in real-time communication made possible with socket.io. Know Me is a single page application made in React. My team and I also used MaterialUI and materialize in creating the design. For the database, we created our own trivia questions with formal research, and stored it in MongoDB. For authorization and authentication, we used Rand-token and Passport. Finally, Know Me was deployed in Heroku. GitHub: KnowMe
